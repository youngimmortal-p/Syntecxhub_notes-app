# 📝 Notes App

A modern, responsive Notes Application built with React.  
It allows users to create, edit, delete, and organize notes with categories, while supporting dark mode and persistent storage.

---

## ✨ Features

- 📝 Add, edit, and delete notes
- 📂 Organize notes by categories
- 🔍 Filter notes by category
- ⏰ Timestamps (created & updated)
- 🌙 Dark mode (saved automatically)
- 💾 LocalStorage support (data persists after refresh)
- 📱 Fully responsive (mobile-friendly)

---

## 🛠️ Built With

- React (Hooks)
  - useState
  - useEffect
  - useRef
- CSS (Modern UI + Responsive Design)
- LocalStorage API
